## نمای کلی بررسی وب‌سایت

بررسی وب‌سایت شامل جمع‌آوری اطلاعات درباره یک وب‌سایت هدف برای شناسایی آسیب‌پذیری‌ها، فناوری‌های استفاده شده و نقاط ورودی احتمالی برای حملات است.

## اطلاعاتی که باید جمع‌آوری شود

- نوع و نسخه وب سرور
- سیستم‌عامل
- زبان‌های برنامه‌نویسی و فریم‌ورک‌ها
- سیستم مدیریت محتوا (CMS)
- افزونه‌ها و پسوندهای شخص ثالث
- ساختار دایرکتوری
- فایل‌ها و دایرکتوری‌های مخفی
- زیردامنه‌ها
- آدرس‌های ایمیل
- فناوری‌ها و کتابخانه‌ها

## تکنیک‌های بررسی وب‌سایت

### بررسی کد منبع وب‌سایت

مشاهده کد منبع صفحه برای یافتن:
- کامنت‌های مخفی
- متادیتا
- فایل‌های JavaScript
- اندپوینت‌های API
- یادداشت‌های توسعه‌دهنده
- نشانه‌های فریم‌ورک

### تحلیل هدرهای HTTP

هدرهای HTTP اطلاعات زیر را آشکار می‌کنند:
- اطلاعات سرور
- زبان برنامه‌نویسی
- جزئیات فریم‌ورک
- هدرهای امنیتی
- پیکربندی کوکی‌ها

### تحلیل Robots.txt

فایل robots.txt می‌تواند موارد زیر را آشکار کند:
- دایرکتوری‌های مخفی
- پنل‌های مدیریت
- مناطق محدود
- مسیرهای غیرمجاز

### تحلیل Sitemap.xml

نقشه‌های سایت اطلاعات زیر را ارائه می‌دهند:
- ساختار کامل سایت
- تمام صفحات ایندکس شده
- تاریخ آخرین تغییر
- صفحات اولویت‌دار

### کپی‌برداری از وب‌سایت

ایجاد کپی‌های محلی از وب‌سایت‌ها برای تحلیل آفلاین با استفاده از ابزارهایی مانند:
- HTTrack
- Wget
- cURL

### شمارش زیردامنه

یافتن زیردامنه‌ها برای گسترش سطح حمله:
- شمارش DNS
- لاگ‌های شفافیت گواهینامه
- کوئری‌های موتور جستجو
- حمله خشن زیردامنه

## ابزارهای بررسی وب‌سایت

### شناسایی فناوری وب

**Wappalyzer**
افزونه مرورگر که فناوری‌های استفاده شده در وب‌سایت‌ها را شناسایی می‌کند.

**BuiltWith**
پروفایل فناوری دقیق وب‌سایت‌ها را ارائه می‌دهد.

**WhatWeb**
ابزار خط فرمان برای شناسایی فناوری‌های وب‌سایت.

### اسکن وب‌سایت

**Nikto**
اسکنر وب سرور که آسیب‌پذیری‌ها و پیکربندی‌های نادرست را تست می‌کند.

**Burp Suite**
پلتفرم جامع تست امنیت برنامه‌های وب.

**OWASP ZAP**
اسکنر امنیت برنامه‌های وب متن‌باز.

### کشف زیردامنه

**Sublist3r**
ابزار Python برای شمارش زیردامنه‌ها.

**Amass**
شمارش عمیق DNS و نقشه‌برداری شبکه.

**DNSdumpster**
ابزار آنلاین بررسی DNS.

### خزیدن وب‌سایت

**Scrapy**
فریم‌ورک Python برای وب اسکرپینگ و خزیدن.

**Beautiful Soup**
کتابخانه Python برای پارس HTML و XML.

**Selenium**
اتوماسیون مرورگر برای محتوای داینامیک.

## استخراج متادیتا

استخراج متادیتا از فایل‌های یافت شده در وب‌سایت‌ها:
- نام نویسندگان سند
- نسخه‌های نرم‌افزار
- تاریخ ایجاد
- تاریخچه ویرایش
- مسیرهای داخلی
- حساب‌های کاربری

**ابزارها:**
- ExifTool
- FOCA
- Metagoofil

## اثرانگشت‌گیری برنامه وب

شناسایی برنامه‌های وب و نسخه‌های خاص:
- تشخیص CMS (WordPress، Joomla، Drupal)
- پلتفرم‌های تجارت الکترونیک
- نرم‌افزار انجمن
- برنامه‌های سفارشی

## اطلاعات WHOIS و DNS

جمع‌آوری داده‌های ثبت دامنه و DNS:
- اطلاعات ثبت‌کننده
- سرورهای نام
- جزئیات تماس
- تاریخ ثبت
- رکوردهای DNS (A، MX، NS، TXT)

## بهترین روش‌ها

- همیشه مجوز مناسب دریافت کنید
- تمام یافته‌ها را مستند کنید
- به محدودیت‌های نرخ و robots.txt احترام بگذارید
- از VPN یا پروکسی برای ناشناس ماندن استفاده کنید
- از اسکن تهاجمی اجتناب کنید
- آسیب‌پذیری‌ها را مسئولانه گزارش دهید
