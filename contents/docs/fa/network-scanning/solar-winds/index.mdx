## SolarWinds چیست؟

SolarWinds ابزارهای مختلف مدیریت و نظارت شبکه را ارائه می‌دهد. در زمینه تست نفوذ، ما بر قابلیت‌های کشف شبکه، نقشه‌برداری و شناسایی تمرکز می‌کنیم.

## ابزارهای کلیدی برای تست نفوذ

### Network Topology Mapper
- کشف خودکار شبکه
- نمودارهای بصری شبکه
- فهرست دستگاه
- ردیابی اتصال

### IP Address Manager
- ردیابی آدرس IP
- مدیریت زیرشبکه
- مدیریت DHCP/DNS
- تشخیص تداخل IP

### Port Scanner
- اسکن پورت TCP/UDP
- تشخیص سرویس
- اندازه‌گیری زمان پاسخ
- تکنیک‌های اسکن متعدد

### ابزار TraceRoute
- کشف مسیر
- تجزیه و تحلیل گام به گام
- زمان پاسخ در هر گام
- عیب‌یابی شبکه

## Network Topology Mapper

### ویژگی‌ها
- اسکن خودکار شبکه
- روش‌های اسکن متعدد
- نقشه‌های توپولوژی بصری
- قابلیت‌های صادرات

### استفاده
```
1. راه‌اندازی Network Topology Mapper
2. انتخاب روش اسکن
3. تعریف محدوده شبکه هدف
4. پیکربندی تنظیمات اسکن
5. شروع کشف
6. مشاهده و صادرات نتایج
```

### روش‌های اسکن
- **SNMP**: کشف دستگاه شبکه
- **WMI**: اسکن دستگاه Windows
- **VMware**: زیرساخت مجازی
- **Hybrid**: روش‌های ترکیبی

## Port Scanner

### اسکن پایه
```
Target: 192.168.1.0/24
محدوده پورت: 1-1024
پروتکل: TCP/UDP
Timeout: 1000ms
```

### ویژگی‌ها
- اسکن سریع
- تشخیص سرویس
- محدوده‌های پورت سفارشی
- صادرات نتایج

## Network Performance Monitor

### قابلیت‌ها
- در دسترس بودن شبکه
- نظارت بر زمان پاسخ
- استفاده از پهنای باند
- سلامت دستگاه

### مفید برای شناسایی
- شناسایی سیستم‌های بحرانی
- نقشه‌برداری وابستگی‌های شبکه
- کشف مسیرهای شبکه
- نظارت بر رفتار هدف

## IP Address Scanner

### ویژگی‌های کشف
- تشخیص هاست زنده
- شناسایی آدرس MAC
- حل نام میزبان
- زمان پاسخ

### پیکربندی اسکن
```
Range: 192.168.1.1-254
روش: ICMP Echo
DNS Lookup: فعال
MAC Resolution: فعال
```

## SNMP Scanner

### جمع‌آوری اطلاعات
- اطلاعات دستگاه
- رابط‌های شبکه
- جداول مسیریابی
- جزئیات سیستم

### نسخه‌های SNMP
- **SNMPv1**: پایه، ناامن
- **SNMPv2c**: رشته‌های انجمن
- **SNMPv3**: احراز هویت و رمزگذاری

### OIDهای رایج
```
توضیحات سیستم: 1.3.6.1.2.1.1.1.0
Hostname: 1.3.6.1.2.1.1.5.0
Uptime: 1.3.6.1.2.1.1.3.0
تماس: 1.3.6.1.2.1.1.4.0
```

## Network Discovery Wizard

### گام به گام
1. **انتخاب روش کشف**
   - ICMP
   - SNMP
   - WMI
   - ترکیبی

2. **تعریف محدوده**
   - محدوده‌های IP
   - زیرشبکه‌ها
   - اعتبارنامه‌ها

3. **پیکربندی گزینه‌ها**
   - مقادیر timeout
   - تلاش‌های مجدد
   - اسکن‌های همزمان

4. **اجرای اسکن**
   - نظارت بر پیشرفت
   - مشاهده نتایج زنده

5. **تجزیه و تحلیل نتایج**
   - لیست دستگاه
   - نقشه توپولوژی
   - صادرات داده

## موارد استفاده شناسایی

### نقشه‌برداری شبکه
```
کشف چیدمان شبکه
شناسایی بخش‌های شبکه
نقشه‌برداری مسیرهای مسیریابی
مستندسازی زیرساخت
```

### کشف دارایی
```
پیدا کردن هاست‌های زنده
شناسایی انواع دستگاه
فهرست‌بندی سرویس‌ها
ردیابی آدرس‌های MAC
```

### پشتیبانی ارزیابی آسیب‌پذیری
```
شناسایی اهداف اسکن
اولویت‌بندی سیستم‌ها
نقشه‌برداری وابستگی‌ها
برنامه‌ریزی مسیرهای حمله
```

## بهترین روش‌ها

### فاز شناسایی
1. با اسکن‌های گسترده شروع کنید
2. محدوده‌های شبکه را شناسایی کنید
3. توپولوژی را نقشه‌برداری کنید
4. یافته‌ها را مستند کنید

### ملاحظات مخفیانه
- نرخ اسکن کند
- زمان‌بندی تصادفی
- اجتناب از الگوهای تشخیص
- در صورت امکان از اعتبارنامه‌های قانونی استفاده کنید

### مستندسازی
- ذخیره نقشه‌های توپولوژی
- صادرات لیست‌های دستگاه
- ثبت زمان‌ها
- یادداشت ناهنجاری‌ها

## ابزارهای خط فرمان

### SNMP Walk
```bash
# استفاده از CLI SolarWinds
snmpwalk -v2c -c public 192.168.1.1
```

### عملیات دسته‌جمعی
```powershell
# مثال اسکریپت PowerShell
Import-Module SolarWinds
Get-NetworkDevices -Subnet "192.168.1.0/24"
```
