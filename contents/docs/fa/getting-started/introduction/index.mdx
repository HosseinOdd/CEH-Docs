به **AriaDocs** خوش آمدید! این قالب یک پایه مدرن، انعطاف‌پذیر و کاربرپسند برای ساخت مستندات جذاب فراهم می‌کند. چه بخواهید یک پایگاه دانش، مستندات پروژه یا بلاگ شخصی بسازید، AriaDocs راه‌اندازی و توسعه محتوای شما را آسان می‌کند.

## فلسفه متن‌باز

AriaDocs با افتخار **متن‌باز** است! 🎉 ما به یک پلتفرم قابل دسترس و مشارکتی اعتقاد داریم که از طریق مشارکت‌های جامعه غنی می‌شود.

<Note title="مشارکت">  
می‌خواهید به ما در بهبود AriaDocs کمک کنید؟ [مخزن GitHub](https://github.com/your-repo) ما را برای شروع ببینید! چه برای پیشنهاد ویژگی‌های جدید یا رفع اشکالات، همه مشارکت‌ها خوشامد هستند.  
</Note>

## نمای کلی پروژه

**AriaDocs** فقط یک قالب ساده برای مستندسازی نیست. این یک **ابزار جامع** است که برای توسعه وب مدرن طراحی شده است. ویژگی‌های اصلی آن عبارتند از:

- **پشتیبانی از Markdown و MDX**: به راحتی مستندات خود را با Markdown بنویسید با قابلیت افزودن کامپوننت‌های تعاملی از طریق MDX.
- **تم‌های قابل تنظیم**: با یک تم مینیمال الهام گرفته از ShadCN طراحی شده که استایل‌دهی آن آسان است.
- **بهینه‌سازی SEO**: هر صفحه برای موتورهای جستجو بهینه شده و دید بهتری را تضمین می‌کند.
- **بلوک‌های کد تعاملی**: قطعه‌های کد زیبا و مخصوص هر زبان برای تجربه خواندن بهتر.

## فناوری‌ها و کتابخانه‌ها

AriaDocs بر روی فناوری‌های پیشرفته تکیه می‌کند تا عملکرد و انعطاف‌پذیری را تضمین کند:

- **Next.js 14** - چارچوب قدرتمند React بهینه‌شده برای تولید.
- **Tailwind CSS** - استایل utility-first برای طراحی سریع و تمیز.
- **Shadcn-UI** - کامپوننت‌های زیبا و قابل دسترس برای نمایش مرتب.
- **next-mdx-remote** - پشتیبانی پویا از MDX برای Markdown تعاملی را فراهم می‌کند.

برای جزئیات بیشتر [مستندات استقرار Next.js](https://nextjs.org/docs/deployment) ما را مشاهده کنید.
