## Types of Attacks

### Active Attacks
Attacks that directly modify or destroy data and system resources.

### Passive Attacks
Attacks focused on gathering information without altering the system.

### Insider Attack
Malicious activities carried out by individuals within an organization.

### Outsider Attack
Attacks initiated by external threat actors with no legitimate access.

## Common Attack Types

### Denial of Service (DoS)
Overwhelming a system or network to make it unavailable to legitimate users.

### Distributed Denial of Service (DDoS)
Coordinated DoS attack from multiple compromised systems simultaneously.

### Brute Force Attack
Systematically trying all possible password combinations until the correct one is found.

### Dictionary Attack
Password cracking using a predefined list of commonly used passwords.

### Rainbow Table Attack
Using precomputed hash values to crack password hashes quickly.

## Network Attacks

- **Sniffing**: Capturing and analyzing network traffic to extract sensitive information
- **Spoofing**: Impersonating another device or user to gain unauthorized access
- **Session Hijacking**: Taking over an active user session to gain unauthorized access
- **ARP Poisoning**: Corrupting the ARP cache to redirect network traffic
- **DNS Spoofing**: Redirecting domain name queries to malicious IP addresses

## Web Application Attacks

- **SQL Injection**: Inserting malicious SQL code into application queries
- **Cross-Site Scripting (XSS)**: Injecting malicious scripts into web pages
- **Cross-Site Request Forgery (CSRF)**: Forcing users to execute unwanted actions
- **File Inclusion**: Exploiting vulnerabilities to include malicious files
- **Command Injection**: Executing arbitrary commands on the host system

## Wireless Attacks

- **Evil Twin**: Fake wireless access point mimicking a legitimate one
- **Wardriving**: Searching for vulnerable wireless networks while mobile
- **Deauthentication Attack**: Forcing devices to disconnect from wireless networks
- **WPS Attack**: Exploiting weaknesses in WiFi Protected Setup
