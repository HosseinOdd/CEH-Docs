## What is SolarWinds?

SolarWinds provides various network management and monitoring tools. In penetration testing context, we focus on network discovery, mapping, and reconnaissance capabilities.

## Key Tools for Pen Testing

### Network Topology Mapper
- Automated network discovery
- Visual network diagrams
- Device inventory
- Connection tracking

### IP Address Manager
- IP address tracking
- Subnet management
- DHCP/DNS management
- IP conflict detection

### Port Scanner
- TCP/UDP port scanning
- Service detection
- Response time measurement
- Multiple scan techniques

### TraceRoute Tool
- Path discovery
- Hop-by-hop analysis
- Response time per hop
- Network troubleshooting

## Network Topology Mapper

### Features
- Automatic network scanning
- Multiple scanning methods
- Visual topology maps
- Export capabilities

### Usage
```
1. Launch Network Topology Mapper
2. Select scanning method
3. Define target network range
4. Configure scan settings
5. Start discovery
```

## Command-Line Tools

### SNMP Walk
```bash
# Using SolarWinds CLI
snmpwalk -v2c -c public 192.168.1.1
```

### Bulk Operations
```powershell
# PowerShell script example
Import-Module SolarWinds
Get-NetworkDevices -Subnet "192.168.1.0/24"
```

## Limitations

- Commercial software (costs)
- Windows-focused
- Can be detected
- Requires proper credentials for deep scanning
- May trigger security alerts

## Alternatives

- **Nmap**: Open-source scanning
- **Angry IP Scanner**: Simple IP scanning
- **NetXMS**: Open-source monitoring
- **Zabbix**: Network monitoring
- **PRTG**: Network monitoring

## Detection and Evasion

### Common Detections
- IDS/IPS signatures
- SNMP query logs
- WMI access logs
- Unusual scan patterns

### Evasion Techniques
- Slow scanning
- Use legitimate accounts
- Scan during business hours
- Blend with normal traffic
